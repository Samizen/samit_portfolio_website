http://www.albinotonnina.com/

The first website I was influenced from was http://www.albinotonnina.com/. This page influenced my home page. I was browsing through different portfolio websites for the coursework when I stumbled upon this. This website is JS heavy and uses animation via the scrolling of the wheel.

What influenced me though was the clean bright vector representation of the page. The icons used was so stylish and elegant. These icons of things you would find in a table of a designer were not clickable though. I wanted to create a system of familiar icons too with clean background and a bold representation of fonts which would give me an impetus on anything I put on display.

The look I managed to achieve is totally different from the website though. I have clean vector icons which are clickable which immediately grabs the attention of my website viewers. The clean background accentuates my icons and is heavily influenced from this landing page. The navigation bar system in place too needed tweaking in my website because I had to put it in top to improve the functionality for all the pages. The figure below shows my website homepage.

The navigation is possible via the navigation bar on top and the icons below. There is also a presence of a footer in my page, again, to establish the element of familiarity within the pages.
